```
Make sure all the checkboxes at the bottom are not checked.
```

```
确认所有的复选框在底部是未选中的... egg...
```

当然我全部读完后心中重新组织一遍会知道正确的应该是

```
确认在底部的所有复选框都没有被选中...
```

但是我心中很别扭。因为每句话我都需要全部读完后在内心重新组织一遍。
